# Daha Ata Sanniya (18 Sanniya) - Interactive Ritual Guide

An interactive, web-based digital catalog exploring the **Daha Ata Sanniya**, the traditional Sri Lankan mask dance ritual used for healing. This project combines cultural heritage with modern web interactivity to provide an educational experience.

## 🌟 Live Demo
View the project here: https://hevan03.github.io/Daha-Ata-Sanniya-Website/

## ✨ Features
* **Interactive 3D Flip Cards:** Each mask features a high-quality visual on the front and a detailed clinical/cultural description on the back.
* **Responsive Grid:** Optimized for viewing on desktops, tablets, and mobile devices.
* **Cultural Accuracy:** Focused on the 18 specific demons (Sanniyas) and their associated ailments.
* **Clean UI:** Minimalist "Night Mode" aesthetic to emphasize the vibrant colors of the traditional masks.

## 🛠️ Built With
* **HTML5** - Semantic structure.
* **CSS3** - Custom animations, 3D transforms, and Flexbox/Grid layouts.
* **JavaScript** - Toggle logic for interactive card flipping.
